#  Pipeline de Dados com Databricks 🟧
<p>
 Este projeto tem como objetivo construir um pipeline de Engenharia de Dados para processamento e análise de transações bancárias utilizando Databricks, PySpark e SQL.
</p>
<p>
  O projeto simula um cenário real onde um banco recebe dados provenientes de sistemas internos em arquivos CSV e necessita transformar esses dados brutos em informações confiáveis para análise de negócio.
</p>
<p>
  A solução foi desenvolvida utilizando a arquitetura Medalhão (Bronze, Silver e Gold), garantindo organização, qualidade, rastreabilidade e disponibilidade dos dados para consumo analítico.
</p>

<h1>Objetivo do Projeto 🎯</h1>
<p>Construir um pipeline completo capaz de: </p>
<p>- Realizar a ingestão de dados</p>
<p>- Armazenar dados brutos preservando histórico</p>
<p>- Aplicar regras de qualidade e transformação</p>
<p>- Criar produtos de dados analíticos</p>
<p>- Disponibilizar informações para tomada de decisão através de dashboards.</p>

<h1>Tecnologias utilizadas ⚙️</h1>
- Databricks <br>
- Arquitetura Medalhão <br>
- Apache Spark <br>
- PySpark <br>
- Python <br>
- SQL <br>
- Power BI
